# Booking Inquiry Form + Booking Email

Interactive walkthrough: configure your Framer booking inquiry form and set up email notifications.

## Deploy to Netlify
1. Drag the entire `booking-inquiry-form/` folder into Netlify's dashboard  
   (or `netlify deploy --dir=booking-inquiry-form`)
2. Copy the deploy URL

## Embed in Framer
Add an Embed component and paste:
```html
<iframe src="YOUR_NETLIFY_URL" width="100%" height="100%" frameborder="0"></iframe>
```

## Controls
| Action | Key |
|--------|-----|
| Next step | → or Enter |
| Previous step | ← |
| Restart | Esc |
| Click hotspot | advances to next step |

## Steps
1. Booking Inquiry Form overview  
2. Open form settings  
3. Select the Contact Form layer  
4. Configure email settings  
5. Add your email address  
6. Proceed with verification  
7. Send Verification Email  
8. Change the email subject  
9. Change the email body  
10. Confirm notification preferences  
11. Select a form field  
12. Set required / optional  
13. Publish changes
